Vibe Coding Curriculum

Day 1: PRD Drafting & UI Engineering

Phase 01
  • PRD Architecting: Learning to draft comprehensive Product Requirements Documents with AI to prevent "scope creep".
  • Rapid UI with Antigravity: Using Antigravity to generate full-stack scaffolds and responsive frontends instantly from your PRD.
  • Initial GitHub Setup: Creating repositories and understanding basic version control for AI-generated code.

Day 2: Logic, Database & Deployment

Phase 02
  • Claude Code Orchestration: Using Claude Code to refactor logic, handle API calls, and build complex feature sets.
  • Cloud Database & Auth: Connecting Supabase for persistent data storage and secure user authentication.
  • GitHub CI/CD Deployment: Mastering automated deployments so every code change goes live instantly.
